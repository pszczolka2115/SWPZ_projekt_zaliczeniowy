Instrukcja:
1. zainstalować Pythona
2. w terminalu wpisać komendę py -m pip install mkdocs mkdocs-material
3. sklonować repozytorium git clone https://github.com/pszczolka2115/SWPZ_projekt_zaliczeniowy.git
4. w terminalu wejść do folderu w którym zostało sklonowane repozytorium
5. uruchomić serwer py -m mkdocs serve 
6. otworzyć w przeglądarce http://127.0.0.1:8000
